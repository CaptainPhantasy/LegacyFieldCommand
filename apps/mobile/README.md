# Legacy Field Command - Field App

This is the offline-first mobile application for field technicians.

## Tech Stack
- **Framework**: React Native (Expo)
- **Language**: TypeScript
- **Local DB**: WatermelonDB (Planned for Phase 1)
- **Backend**: Supabase

## Key Features (Phase 1)
- Offline Job List
- Guided Workflows (Gates)
- Photo Capture & Annotation
- Sync with Supabase

## Running Locally
```bash
npm install
npm start
```
