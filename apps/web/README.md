# Legacy Field Command - Admin & CRM

This is the web-based Admin Dashboard and CRM for Legacy Field Command.

## Tech Stack
- **Framework**: Next.js 15 (App Router)
- **Language**: TypeScript
- **Styling**: Tailwind CSS
- **Database**: Supabase (client-side & server-side)

## Key Features (Phase 0/1)
- User Management (Techs, Estimators, Admins)
- Job List & Pipeline View
- Reviewing Field Data (Photos, Gates)

## Running Locally
```bash
npm run dev
```
